# Quellenvorlage

Dieser Ordner dient nur als Vorlage fuer lokale Quellordner.

Die echten Originalfolien sollen lokal unter `sources/<fach-id>/` liegen, aber wegen Urheberrecht und GitHub Pages nicht ins oeffentliche Repo gepusht werden.

Empfohlene lokale Struktur:

- `sources/rabs/`
- `sources/mathe-2/`
- `sources/pum/`

Jeder Fachordner sollte eine `manifest.json` und die zugehoerigen PDF-Dateien enthalten.
